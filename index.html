Chatvis.com is a tool we use at Bruna & Bruna for idea generation and concepting. If you like it, please use it and give us feedback so we can improve it.

  • You can create a new board or visit a shared link that brings you to an existing board.
  • You can chat ideas and drag them to group them with other ideas.
  • Ideas are placed where your mouse is.
  • You can make them bigger, smaller or delete them.
  • You can share the URL in the locationbar of your browser with other people so they can work on the same board.

WARNING: Chatvis.com is in BETA, so we take absolutely no responsibily for your data. If you create a board it could be there the next day, the next week, but it could also be lost in an hour. This is because were are still experimenting with the tool. The code and database are changed constantly.

Please, send your feedback to chatvis@brunabruna.nl
